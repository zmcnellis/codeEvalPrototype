# Coderx Prototype

This is the repo for the Coderx website.

## Installation
```sh
$ git clone https://github.com/zmcnellis/codeEvalPrototype.git
$ cd codeEvalPrototype
$ npm install
$ bower install
$ npm start
```
View at http://localhost:8080
