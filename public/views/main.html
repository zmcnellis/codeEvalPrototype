This application is a prototype for the development of an online code quality evaluator. It is part of an ongoing research project at Clemson University, with a focus on assisting early learners in developing a cognitive model of computation that supports future success in the academic curriculum. This effort is motivated through the construction of a generic program representation for automated metric computation.
